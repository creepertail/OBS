# 測試數據生成指令

這個腳本會自動生成完整的測試數據，供前端開發和測試使用。

## 使用方式

在後端專案根目錄執行：

```bash
npm run seed
```

## 注意事項

1. **資料庫連接**：確保你的 `.env` 文件中有正確的資料庫連接設定
2. **清空數據**：此腳本會先清空現有的測試數據，請謹慎使用
3. **密碼加密**：所有帳號密碼都使用 bcrypt 加密存儲
4. **執行時機**：建議在開發或測試環境執行，不要在生產環境使用

## 環境變數

確保 `.env` 文件包含以下設定：

```env
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=your_password
DB_DATABASE=OBS
```

## 問題排查

如果執行失敗，請檢查：
1. 資料庫是否正在運行
2. 環境變數是否正確設定
3. 資料庫連接權限是否足夠
4. 資料表是否已經建立（執行 migration）

## 清除數據

如果需要重新生成數據，直接再次執行 `npm run seed` 即可，腳本會自動清空舊數據。
